* 순서
    * dependency 추가
    * application.properties 설정

* pom.xml

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-devtools</artifactId>
</dependency>
```

* application.properties

```
spring.devtools.livereload.enabled=true
spring.freemarker.cache=false
```
